# Mercad-o-das-Carnes-IA
Uma camada de Inteligência Artificial para gestão de frigoríficos e açougues
 O sistema foca em dois pilares: Eficiência Operacional e Fidelização de Clientes.

🚀 Funcionalidades (Protótipo)

1. Gestor de Balcão (Agente de Eficiência)

Análise de Estoque: Identifica itens com baixo giro ou validade próxima.

Sugestões de Combos: Cria ofertas automáticas para evitar desperdício de produtos sensíveis.

Alertas Críticos: Notifica o gestor sobre perdas iminentes.

2. Concierge de Marketing (Agente de Fidelização)

Perfis de Compra: Segmenta clientes por comportamento (ex: Churrasqueiro, Cozinha Semanal).

Ofertas Personalizadas: Gera mensagens de WhatsApp baseadas no histórico de compra.

Cross-selling Inteligente: Sugere acompanhamentos (carvão, temperos) para aumentar o ticket médio.

🛠️ Tecnologias

Frontend: HTML5, Tailwind CSS.

Lógica: JavaScript (Simulação de Processamento de IA).

Design: Responsivo e focado em UX de gestão.

Projeto desenvolvido por Pâmela para apresentação de solução de IA aplicada ao varejo de carnes.
